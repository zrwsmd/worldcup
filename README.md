# ⚽ 2026世界杯球星阵容展示

一个精美的2026世界杯球星阵容展示页面，采用动态战术布阵展示各国家队的首发阵容和球员信息。

## ✨ 功能特性

- 🎯 **7支豪门球队**：西班牙、法国、阿根廷、英格兰、巴西、德国、葡萄牙
- ⚽ **真实球员照片**：通过维基百科API动态加载球星真实照片
- 📐 **战术阵型展示**：4-3-3、4-2-3-1等经典阵型可视化
- 🎨 **精美UI设计**：足球场背景、位置色彩区分、悬停动画效果
- 📊 **球队统计数据**：平均年龄、国家队出场数、进球数、球队身价
- 🔄 **流畅切换**：一键切换不同国家队阵容
- 📱 **响应式设计**：支持各种屏幕尺寸

## 🐛 常见问题

**Q: 为什么有些球员照片不显示？**  
A: 可能是维基百科上没有该球员的照片，或者维基百科标题配置不正确。系统会自动显示金色背景作为替代。

**Q: 为什么直接打开文件照片加载不出来？**  
A: 浏览器的CORS安全策略限制了 file:// 协议访问API。建议使用本地HTTP服务器运行。

**Q: 如何更换球员照片？**  
A: 修改 `wikiTitles` 对象中对应球员的维基百科标题，或者直接在球员数据中添加 `photo` 字段指定图片URL。
